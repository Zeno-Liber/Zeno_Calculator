# Zeno Calculator

A simple calculator script meant to be a personal learning project.

---

# Features

* Addition
* Subtraction
* Multiplication
* Division

---

## Installation

Clone the repository:

```bash
git clone https://github.com/Zeno-Liber/Zeno_Calculator.git
cd zeno_calculator
```

---

## Usage

Run the program:
```bash
python zeno_calculator.py
```

The program will then prompt the user to enter two numbers and select from a list of options on the screen.
Then the desired calculation will be performed and displayed. The program will continue prompting the user
until they exit the program.
